Steps to Reproduce
Create new python project
Create new venv, activate it and install pip3 install python_telegram_bot
Write following code in main.py
import logging
from telegram import Update
from telegram.ext import ApplicationBuilder, CallbackContext, CommandHandler`

logging.basicConfig(
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    level=logging.INFO
)

async def start(update: Update, context: CallbackContext.DEFAULT_TYPE):
    await context.bot.send_message(chat_id=update.effective_chat.id, text="I'm a bot, please talk to me!")

if __name__ == '__main__':
    application = ApplicationBuilder().token('TOKEN').build()

    start_handler = CommandHandler('start', start)
    application.add_handler(start_handler)

    application.run_polling()
Update your telegram TOKEN
Run the code python3 main.py
Expected behaviour
Code should run and whenever user enters some message then bot should respond.

Actual behaviour
Code is not running. it shows an error
ImportError: cannot import name 'ApplicationBuilder' from 'telegram.ext'
